Programming_C_Summative

Project 1. Smart Traffic Light Controller with Embedded Simulation and PCB Design

Controls two intersections with LEDs and buttons.
Green light timing adjusts automatically based on car count.
Uses millis() for smooth, non-blocking operation.
Includes Tinkercad simulation and EasyEDA PCB design.
Serial monitor shows real-time traffic info.

Project 2: Linux Server Health Monitoring and Alert Automation Script

Monitors CPU, memory, disk usage, and active processes.
Sends alerts if thresholds are exceeded.
Logs activity with timestamps.
Includes simple menu to check system, view logs, and exit.
Runs continuously using loops and sleep intervals.

Project 3: Course Performance and Academic Records Analyzer in C

Stores student records dynamically using structures and memory allocation.
Supports Add, View, Update, Delete (CRUD) operations.
Calculates average GPA, top student, and course performance.
Saves/loads records from files for persistence.
Simple menu interface for user interaction.

Project 4: Data Analysis Toolkit Using Function Pointers and Callbacks

Processes numeric datasets with function pointers and callbacks.
Supports sum, average, min/max, filter, transform, sort, and search.
Datasets managed with dynamic memory (malloc/realloc/free).
Can load/save data from files.
Menu interface allows choosing operations at runtime.

Project 5: Multi-threaded Web Scraper

Fetches multiple web pages in parallel using POSIX threads.
Saves each page to a separate text file.
Handles errors if a URL is unreachable.
Demonstrates simple parallel processing with threads.
